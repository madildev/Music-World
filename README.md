# Music-World

This application created in vue.js in an music info application where you can find information about all your favourite songs

## Requirements

For this application to run you need Node.js install on your system

## How to use

You can use the applciation by: 
* First, run the application
* Next, search the song you want by the Singer's Name/Gener/Song Name
* You can also select from the trending songs
